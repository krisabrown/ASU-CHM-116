# ASU CHM 116 Final Grade Calculator

A Vite React app for estimating CHM 116 final course grades.

## Local setup

```bash
npm install
npm run build
npm run preview
```

## GitHub Pages

This project is configured for the repository name `ASU-CHM-116` and deploys to:

```txt
https://krisabrown.github.io/ASU-CHM-116/
```

In GitHub, go to Settings > Pages and set Source to GitHub Actions.
